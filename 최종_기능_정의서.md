# 🚀 ANG (Admin Next Generation) 최종 기능 정의서

## 1. 프로젝트 개요
ANG 시스템은 대학 행정 실무를 지원하기 위한 차세대 지능형 행정 관리 솔루션입니다. Local LLM(Ollama)을 활용한 AI 비서 서비스와 HWP 문서 처리 자동화 기술을 결합하여, 행정 업무의 효율성을 극대화하고 부서 간 협업을 원활하게 지원합니다.

## 3. 기술적 특이사항
- **Local AI Infra**: 외부 클라우드 의존 없이 서버 내부의 Ollama를 활용하여 보안성이 뛰어난 AI 환경 구축.
- **HWP Bridge**: Windows 환경의 한글(HWP) 자동화를 API화하여 Linux 환경의 백엔드와 유연하게 연동.
- **Hierarchical Access Control**: 단순 부서 단위가 아닌 상위 부서(L2) 공유 로직을 통한 실제 행정 체계 반영.
- **Modern Web Interface**: React 기반의 반응형 UI 및 최신 브라우저 API(File System Access 등) 적용.
